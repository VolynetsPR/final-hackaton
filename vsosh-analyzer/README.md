#  Аналитика ЗЭ ВсОШ по информатике 2024 года

## Запуск
```bash
python3 -m pip install -r requirements.txt
python3 init_db.py
python3 app.py
```

Откройте `http://127.0.0.1:5000/`

## Важные файлы
- `init_db.py` — полностью пересоздаёт SQLite БД из приложенных HTML/CSV файлов
- `app.py` — главное Flask-приложение
- `school_utils.py` — отдельный файл с функцией `get_school(full_name)`
- `data/olympiad.sqlite3` — созданная база данных
- `templates/standings.html` — единственный шаблон таблицы результатов; данные попадают только при рендеринге
- `static/standings2.css` — исходный стиль таблиц результатов

## Проверка
1. Остановить приложение
2. Удалить `data/olympiad.sqlite3`
3. Запустить `python3 init_db.py`
4. После создания БД можно удалить исходные HTML/CSV из `uploads/`
5. Запустить `python3 app.py`

## На текущий момент
### Реализовано
- Статистика по регионам по 4 критериям с переключателями и корректными диапазонами мест.
- Форма генерации результатов: тур, класс, оформление, регион, школа, момент времени.
- Обработка отсутствующих/некорректных моментов времени.
- Подсветка победителей и призёров через классы `row-win-*` и `row-prize-*`.
- Личные страницы участников с полной сводкой.
### Не реализовано
- Статистика по школам Москвы и Санкт-Петербурга.
- Графики баллов и места: первый тур, второй тур, оба тура.
